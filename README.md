# Yardstick | Full-stack Developer Internship Assignment

This is Full-stack Developer Internship at Yardstick Assignment submission

## Tech used

- nextjs
- nextjs server actions
- mongodb
- tailwindcss
- typescript

## Installation

1. Clone the repo:

   ```sh
   git clone git@github.com:Sahil-4/yardstick-assignment.git sahil4AssignmentSubmission
   ```

2. Change directory:

   ```sh
   cd yardstick-assignment
   ```

3. Install required deps:

   ```sh
   npm install
   ```

## Start

4. Start dev server

   ```sh
   npm run dev
   ```
